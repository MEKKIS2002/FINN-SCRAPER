Cloudflare Worker Setup

Denne versjonen gir deg mer konservativ confidence, notifikasjons-cooldown, bedre profit-beregning og mer robust håndtering av bilder og duplikater.

01
Installer Wrangler og opprett prosjekt
Installer Cloudflare CLI, logg inn og opprett KV namespace.
npm install -g wrangler wrangler login mkdir finn-hunter && cd finn-hunter wrangler init --no-bundle wrangler kv:namespace create FINN_HUNTER_KV
02
wrangler.toml
KV binding og cron-trigger for automatisk skanning annenhver time.
name = "finn-hunter" main = "worker.js" compatibility_date = "2024-01-01" [[kv_namespaces]] binding = "KV" id = "DIN_KV_NAMESPACE_ID_HER" [triggers] crons = ["0 */2 * * *"]
03
worker.js — komplett kode
RSS-henting, Claude Vision-analyse, deep research, confidence-normalisering og webhook-varsler.
04
Deploy og sett secrets
Sett API-nøkler som secrets og deploy workeren.
wrangler secret put ANTHROPIC_API_KEY wrangler secret put WORKER_SECRET wrangler secret put NOTIFICATION_WEBHOOK_URL wrangler deploy # Test: curl https://finn-hunter.din.workers.dev/scan \ -H "Authorization: Bearer DIN_SECRET"
